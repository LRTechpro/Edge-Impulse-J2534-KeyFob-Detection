# Contributing

Contributions that improve reproducibility, calibration, post-processing, or embedded performance are welcome.

Please:

1. Describe the hardware and model version used.
2. Keep raw observations separate from interpretation.
3. Do not commit credentials, private URLs, proprietary calibration data, or vehicle-identifying information.
4. Document changes to thresholds, camera resolution, preprocessing, or model weights.
5. Include representative serial output or test data for behavior changes.
